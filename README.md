# What is Automation Get Point and Redeem Reward Telkomsel (Login Twitter)? 

The tool that can get your point and redeem reward telkomsel with automation by login Twitter.

Usage:

    1. Download ChromeDriver

    2. Install Python

    3. Install Pip

    4. Install Selenium

    5. Run the Script
